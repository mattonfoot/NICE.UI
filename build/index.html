Introducing NICE Bootstrap.

Need reasons to love NICE Bootstrap? Look no further.

By us, for us.

Built at NICE by us, NICE Bootstrap utilizes LESS CSS, is compiled via Node, and is managed through GitHub to help us do awesome stuff on the web.

Made for everyone.

NICE Bootstrap was made to not only look and behave great in the latest desktop browsers (as well as IE7!), but in tablet and smartphone browsers via responsive CSS as well.

Packed with features.

Specialist layouts, a 12-column responsive grid, dozens of components, interaction components, typography and form controls.
