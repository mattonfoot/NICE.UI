Semantic information

In some circumstances it is necessary to distinguish certain items of information in a specific descriptive style

Definitions of terms

Wrap <dfn> around any information that is a defining instance of a term.

The Internet is a global system of interconnected networks that use the Internet Protocol Suite (TCP/IP) to serve billions of users worldwide.

WWW
The World-Wide Web (WWW) is a system of interlinked hypertext documents accessed on the Internet.

Editorial history

Sometimes it is important to represent previous thoughts and edits made on the current text.

Wrap <del> around any information that has been removed from the current text and wrap <ins> around any information that has been inserted into the current text since it was first published.

Lorem ipsum dolor sit amet, consecteturadipiscing elit. Praesent feugiat lacinia urna nec egestas. Donec.

Positional formatting

Text that should be displayed, for typographic reasons, higher or lower, and smaller, than the main span of text should be wrapped with <sup> or <sub> respectively.

This text is superscripted

This text is subscripted
